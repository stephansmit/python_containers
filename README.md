# Singularity container for Python scripts

## Build Container
~~~
sudo singularity build python_containers.sif Singularity
~~~

## Pull Image
~~~
singularity pull shub://stephansmit/python_containers
~~~

### Run Image
~~~
singularity run python_containers test.py
~~~
